核心功能

为运维而生的管理面板

从服务器部署到用户计费，FRP Panel 提供完整的内网穿透管理解决方案

多服务器集群

集中管理所有 frps 服务器，支持一键部署、配置同步、实时监控。Dashboard API 和 HTTP Plugin 双通道通信。

一键 SSH 部署

输入服务器 IP 和 SSH 凭证，自动完成 frp 下载、配置生成、systemd 注册、健康检查全流程。

用户与权限

多级 RBAC 权限体系，支持邀请码注册、API 密钥管理、账户余额系统。

流量计费

实时流量统计、套餐配额管理、超额自动处理。支持月付/季付/年付，余额/支付宝/微信支付。

实时监控

CPU/内存/磁盘/网络/连接数全方位监控。10 秒级状态更新，Dashboard API 轮询 + Plugin 事件推送。

配置生成

自动生成 frps/frpc 配置。支持 TCP/UDP/HTTP/HTTPS/STCP/XTCP 六种代理类型。

常见问题

有疑问？看看这里

直接使用 frp 需要手动编写配置文件、管理进程、监控状态。FRP Panel 提供可视化管理界面，支持一键部署服务器、自助创建代理、实时监控、流量统计、用户计费等功能，大幅降低运维成本。
支持 TCP、UDP、HTTP、HTTPS、STCP（安全 TCP）、XTCP（P2P 穿透）六种代理类型。HTTP/HTTPS 类型支持自定义域名和子域名。
完全支持自部署。面板需要 Go 环境和 SQLite 或 PostgreSQL，可选 Redis。提供 Docker Compose 一键部署方案。frps 服务器需要 Linux，面板通过 SSH 自动部署。
面板每 5 分钟通过 frps Dashboard API 拉取各代理的流量数据，计算增量后写入数据库。数据来源是 frps 自身的流量计数器，准确可靠。
内置余额支付，同时支持接入支付宝和微信支付。支付配置在管理端系统设置中完成。
