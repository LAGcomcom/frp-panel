# 在线更新接入

公开版面板可连接独立的 FRP Panel Update Center，用于检查更新和可选的实例存活统计。公开版不需要许可证或激活码，更新中心不可阻止基础面板功能。

```yaml
update:
  center_url: "https://08642.xyz"
  instance_id: "" # 留空时从本机 Ed25519 公钥生成稳定 ID
  panel_version: "1.0.0"
  panel_domain: "https://panel.example.com"
  heartbeat_enabled: true
  heartbeat_interval: 5s
  control_public_key: "HU5iQjEL7v24v5aK0K+gHTctvWorW+iLq5bhpXf9lSU="
  identity_key_file: "data/update-identity.key"
  lease_cache_file: "data/update-lease.json"
  bootstrap_cache_file: "data/update-bootstrap.json"
  bootstrap_urls:
    - "https://raw.githubusercontent.com/owner/bootstrap/main/bootstrap.json"
  anonymous_statistics: false # 旧配置兼容项，新部署使用 heartbeat_enabled
```

启用心跳后，公开版面板会生成独立 Ed25519 实例身份，通过限流的公开入口提交签名证明完成首次登记；之后每次心跳使用实例私钥签名并携带一次性随机数。更新中心拒绝错误签名和重复随机数。许可证产品仍使用独立的 enrollment token 和租约接口。

`heartbeat_enabled` 默认关闭，只有管理员明确开启才会上报。上报内容仅包含实例 ID、自报域名、面板版本、操作系统、架构和最后在线时间。最短周期为 5 秒，避免错误配置造成请求风暴。

不会上报用户、订单、代理、流量、访问者、服务器密码或许可证内容。关闭心跳或更新服务不可用都不会阻止登录、隧道、支付、备份和数据导出。

私有 GitHub 仓库的 Token 必须只配置在更新中心，不能放入面板配置或浏览器前端。授权产品的签名租约和自动激活属于独立产品路径，不用于限制公开版。
