# 在线更新接入

授权面板支持连接独立的 FRP Panel Update Center。该功能默认关闭，只有管理员明确配置后才会登记实例。

```yaml
update:
  enabled: true
  center_url: "https://updates.example.com"
  instance_key: "由更新中心配置的实例密钥"
  panel_version: "1.0.0"
  panel_domain: "https://panel.example.com"
  control_public_key: "控制中心启动日志输出的 Ed25519 公钥"
  identity_key_file: "data/update-identity.key"
  lease_cache_file: "data/update-lease.json"
  bootstrap_cache_file: "data/update-bootstrap.json"
  bootstrap_urls:
    - "https://raw.githubusercontent.com/owner/bootstrap/main/bootstrap.json"
  anonymous_statistics: false
```

面板启动后会生成独立 Ed25519 实例身份，并在后台单例续租。短期租约控制私有更新、云告警和高级报表；失联超过宽限期只关闭这些在线增值功能，不影响基础登录、隧道、备份和数据导出。

`instance_key` 只用于首次登记。续租和私有下载使用实例签名、随机数和重放保护。`anonymous_statistics` 单独控制可选统计，不控制核心租约。

不会上报用户、订单、代理、流量、访问者、服务器密码或许可证内容。更新服务不可用不会阻止面板后台正常工作。

私有 GitHub 仓库的 Token 必须只配置在更新中心，不能放入面板配置或浏览器前端。当前版本提供管理员确认后的更新包入口，不会自动替换运行中的面板程序。
